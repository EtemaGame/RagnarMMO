# Changelog

## [0.0.2-beta] - 2026-05-12

### Added

- Updated the mod version to `0.0.2-beta`.
- Reworked the project README to better describe the current beta state, roadmap and contribution needs.
- Added release notes in English for this beta.

### Changed

- Refined the public project description to match the current scope more accurately.
- Cleaned up the mod metadata description to reflect the RPG-focused direction more clearly.
- Removed placeholder mobs and their related references from code, assets and localization.
- Cleaned the creative tab so only the currently supported mobs remain visible.

### In Progress

- Weapon art and mob art are still being improved.
- Balancing for combat, progression and rewards is still being tuned.
- Additional content and polish will continue to land in future beta builds.

### Notes

- The current weapons and mobs have been reworked compared to the earlier placeholder pass.
- The project is actively looking for a modeler to help with future monster and weapon work.
- One helper is already contributing, and more collaboration is welcome as the content set grows.
