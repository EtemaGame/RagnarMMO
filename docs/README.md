# Docs Index

The `docs` folder was cleaned to keep only living documentation that still helps maintain or extend the project.

## Keep these documents

- `system-extension-guide.md`
  - practical guide for extending jobs, skills, manual mobs, and content data
- `design/mob-datapack-spec.md`
  - external contract for manual mob datapack authoring
- `design/mob-taxonomy-spec.md`
  - semantics of `race`, `element`, and `size`
- `design/mob-stat-derivation-spec.md`
  - semantic meaning of `ro_stats` and final combat stats
- `rework/`
  - active Pre-Renewal first-job rework notes, policy, and canonical skill matrix

## Removed from docs

The old planning notes, migration slices, readiness gates, temporary coexistence notes, and other step-by-step transition documents were removed because they no longer serve as primary documentation for the current codebase.

If a future redesign starts, add new docs as focused living references, not as another long migration diary.
