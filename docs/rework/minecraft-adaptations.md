# Minecraft Adaptations

This document records the small set of non-RO primitives allowed when translating Pre-Renewal skills into Minecraft behavior.

## Data Grammar

Use existing skill JSON `level_data`. Do not introduce another skill tuning format.

Standard active-skill keys:

| Key | Meaning |
| --- | --- |
| `sp_cost` | SP cost at this skill level. |
| `cast_time_ticks` | Cast time before execution. |
| `cast_delay_ticks` | After-cast delay or global skill delay. |
| `cooldown_ticks` | Per-skill cooldown. |
| `damage_percent` | Damage multiplier in percent. |
| `hit_count` | Number of logical hits. |
| `duration_ticks` | Buff, debuff, or ground effect duration. |
| `success_chance` | Generic success chance when the skill needs a roll. |
| `status_chance` | Status application chance. |
| `aoe_radius` | Area radius in blocks. |
| `knockback_strength` | Explicit knockback strength when knockback is part of the design. |

Minecraft adaptation keys:

| Key | Meaning |
| --- | --- |
| `projectile_velocity_mult` | Multiplies projectile speed for adapted ranged skills. |
| `projectile_gravity_mult` | Multiplies projectile gravity/drop. |
| `projectile_spread_mult` | Multiplies projectile inaccuracy/spread. |
| `aggro_bonus` | Extra threat generated by a skill. |
| `reveal_radius` | Reveal radius for hidden/cloaked/invisible targets. |
| `undead_multiplier` | Undead-specific damage or healing conversion multiplier. |
| `mob_pull_bonus` | Extra mob pull or attention behavior. |
| `vendor_buy_discount` | Merchant buy discount. |
| `vendor_sell_bonus` | Merchant sell bonus. |

## Derived Stats To Add

These fields should live on the derived-stat layer so skills do not need local hacks:

| Field | Purpose |
| --- | --- |
| `projectileVelocityMult` | Buff or passive modifier for projectile speed. |
| `projectileGravityMult` | Buff or passive modifier for projectile drop. |
| `projectileSpreadMult` | Buff or passive modifier for projectile accuracy/spread. |
| `castInterruptResist` | Resistance to cast interruption. |
| `skillStatusPower` | Shared modifier for status success or status strength. |
| `threatGenerationBonus` | Shared modifier for aggro/threat output. |
| `revealRadiusBonus` | Shared modifier for reveal radius. |

These stats do not replace RO stats. They are translation hooks for cases where Minecraft has no direct equivalent.

## Archer Decisions

| Skill | Adaptation |
| --- | --- |
| Owl's Eye | DEX passive. No projectile or damage side effect. |
| Vulture's Eye | Accuracy/range passive through HIT, less gravity, slightly more projectile speed, and less spread. No direct damage bonus. |
| Improve Concentration | Explicit DEX/AGI buff plus separate reveal radius. |
| Double Strafing | Always two logical hits, bow-only unless a later design note opts into crossbows. |
| Arrow Shower | Area cleanup/control. Knockback must be explicit and should not dominate single-target damage identity. |
